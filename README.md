### Pet todo app with mongo db

**Server:**

```
- MongoDB
```

**Client:**

```
- Vue
- Nuxt
- Pinia
```

## How to run:

- clone repo
- run `docker-compose -f todo-app.yml up -d`
- go to `localhost:3000`
